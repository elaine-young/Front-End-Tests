Jquery Test

Moonpig needs an accordian for the site, but we want it to be light weight and small. Without using a plugin, hand write some jQuery to create an accordian effect.

  1. All panels of links should be hidden from view on load, except the first panel.
  2. When a user clicks on the title, the respective panel should expand, and the other panels should shrink
  3. If a user clicks on a title with an already expanded panel, that panel shrinks so that none of the panels are visible
  4. BONUS TASK! - The script will not run if you are on a mobile sized screen ;)
